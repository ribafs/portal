Instalar o Laravel com o Composer Globalmente no Linux


Abrir Terminal e executar
composer global require "laravel/installer"

Agora para criar os projetos se coloca o laravel no path assim (Ubuntu e derivados):
echo 'export PATH="$PATH:$HOME/.config/composer/vendor/bin"' >> ~/.bashrc

Fechar o terminal e abrir novamente para surtir efeito

Teste assim:
laravel new blog

Executar o aplicativo com o servidor interno do PHP
php artisan serve

Ver pelo navegador
http://127.0.0.1:8000

Outra forma de instalar o laravel, desta vez um laravel por projeto, usando o composer

composer create-project --prefer-dist laravel/laravel blog

Ajuste as permissões, se em linux e execute

Usando Apache ou outro servidor web
http://localhost/blog
