Introdução ao Framework CakePHP

O CakePHP torna a criação de aplicativos web mais simples, rápida enquanto requer uma menr códificação. Um moderno framework para PHP 7 oferece uma camada de acesso a bancos de dados flexível e um poderoso sistema de scaffolding que torna a criação de pequenos ou complexos sistemas mais simples, fácil e agradável.

Construção rápida e sólida com CakePHP.

Todos os recursos necessários para a criação de aplicativos, inclusive plugin de ACL. Ainda por cima criei meu próxio plugin de controle de acesso com recursos gráficos. Em outros frameworks populares este recurso é bem complicado.

Está atualmente na versão 3.4
