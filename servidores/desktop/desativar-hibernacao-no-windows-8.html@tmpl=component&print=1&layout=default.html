Desativar Hibernação no Windows 8+

Imprimir

Quando tentamos montar uma partição do Windows 8 ou superior no Ubuntu ou derivados, então devemos desativar a Hibernação:

Presione a Tecla Windows + X e execute como Administrador o Prompt de comando.

Com o Prompt aberto digite:

powercfg -h off

e pressione enter.

Agora a hibernação no Windows está desativada.
