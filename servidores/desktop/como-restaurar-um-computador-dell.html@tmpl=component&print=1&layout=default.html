Como Restaurar um Computador Dell

Aviso Importante: esta ação apagará todos os dados na(s) unidade(s) de destino. Então se necessário e possível faça backup antes deste procedimento.

Problema e Solução

Este é mais um ótimo serviço que a Dell presta a seus clientes, além dos oferecidos para quem cria uma conta em seu site.

Em caso de problema irrecuperável no sistema operacional Windwos ou Linux, por uma falha no sistema ou troca de HD, então podemos usar o software da Dell OS Recovery Tool, que gera um disco de boot (Pendrive) contendo o Windows para dar boot com esse pendrive e reinstalar o sistema Windows ou Linux além de reconfigurar a BIOS como veio de fábrica.

No meu caso, havia removido todas as partições originais e criado outras para usar linux, assim como havia desabilitado o UEFI e Secure Boot na BIOS.

Ao final ele apagou todas as partições e criou apenas uma de 1TB contendo o Windows 10 e deixou a BIOS com UEFI e Secure Boot habilitado.

Requisitos

Acesse:

https://www.dell.com/support/article/br/pt/brbsdt1/sln299044/como-fazer-download-e-usar-o-dell-os-recovery-tool-do-microsoft-windows?lang=pt

- Anote a etiqueta do computador

- Acesse https://www.dell.com/support/home/br/pt/brbsdt1/drivers/osiso/win

Entre com a TAG/Etiqueta do computador

Digite abaixo o código de segurança

Faça o download da image ou imagens ISO e do software de gravação no pendrive Dell OS Recovery Tool

- Instale, como administrador, o Dell OS Recovery Tool em outro computador com Windows 7 ou superior com .NET Framework 4.5 ou neste computador e grave a ISO num pendrive de 8GB.

Veja as instruções aqui, de acordo com o sistema e a versão:

https://www.dell.com/support/article/br/pt/brbsdt1/sln299044/como-fazer-download-e-usar-o-dell-os-recovery-tool-do-microsoft-windows?lang=pt

- Reiniciar o computador onde deseja restaurar com o pendrive conectado e teclar F12 para mudar a sequência de boot, se necessário

- Na tela “Escolha uma opção”, selecione Solucionar problemas e, em seguida, selecione Recuperar de uma unidade.
