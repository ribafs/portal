Instalando Impressora HP Deskjet Ink Advantage 2676 Wi-Fi no Linux Mint 19.1

Instalar em um micro de rede wi-fi para imprimir pelo wi-fi e não via cabo

Download do HPLIP 3.19.6
https://developers.hp.com/hp-linux-imaging-and-printing/gethplip

Orientações
https://developers.hp.com/hp-linux-imaging-and-printing/install/install/index

Instalação

cd /home/ribafs
mkdir hp
cd hp

Mover o arquivo baixado para /home/ribafs/hp

Executar:
sudo sh hplip-3.19.6.run

Após instalar reinicie a máquina

Quando voltar conecte a impressora (temporariamente) pelo cabo USB no micro

Execute pelo menu - HP Device Manager

Clique na opção Wireless ... e então abaixo em Next

Siga os demais passos até concluir.

Ao final ele instala duas impressoras, uma local como default e outra Wi-fi.
Eu troquei a Wifi para ser a Default.
