Compactar multivolumes no Linux

Instale os pacots abaixo


sudo apt-get -y install unace rar unrar zip unzip p7zip-full p7zip-rar sharutils uudeview mpack arj cabextract file-roller xarchiver

Estes oferecem suporte via Nautilus/Nemo
.rar ou .7z

Agora pode compactar multivolume através do modo gráfico com o gerenciador de arquivos Nautilus ou Nemo.
