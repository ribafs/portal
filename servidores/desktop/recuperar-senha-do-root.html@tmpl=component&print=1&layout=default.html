Recuperar senha de root no Linux


Mesmo em distribuições baseadas no Ubuntu, onde se usa o sudo é importane conceder senha ao root, pois pode acontecer de precisar, como quando o sistema corrompe e temos de acessar no modo recovery.

Este pequeno tutorial é para quando esquecemos a senha do root.

Dar boot com live CD

Acessar o terminal após o sistema ser carregado

Descobrir a partição raiz em que está o Linux instalado com

sudo su

fdisk -l

Exemplo: /dev/sda5

mkdir teste
mount /dev/sda5 teste

chroot teste

passwd root

Criar novo usuário se precisar
adduser novo_user

umount teste

reboot

Acessar linux com novo_user
