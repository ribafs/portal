Alternativas para o Prompt do Windows


O prompt do Windows tem várias limitações (pelo menos até a versão 7):
- Não fecha com ALT+F4
- Não maximiza a janela
- Não suporta SSH
- Não suporta Git

As alternativas abaixo não têm estas limitações

CMDER - https://github.com/cmderdev/cmder/releases/
Full e mini

PowerCMD - http://www.powercmd.com/

ColorConsole - http://www.softwareok.com/?seite=Freeware/ColorConsole

ConEmu - https://conemu.github.io/

Console - http://sourceforge.net/projects/console/

GIT - https://git-scm.com/
