Criar Atalho para Desligar, Reniciar e etc o Computador

No Ubuntu e derivados atualmente a combinação Ctrl+Alt+Del não mais reinicia o compitador.

Mas existe uma alternativa de se criar uma tecla de atalho customizada que execute o comando:

gnome-session-quit --power-off

E atribuir para a combinação Ctrl+Alt+Del

Assim novamente poderemos desligar, reiniciar, entre outras com Ctrl+Alt+Del.
