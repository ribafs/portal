Hospedagem Grátis para PHP e MySQL

Como é uma hospedagem para php e mysql, então pode ser usada para a criação de sites com Joomla, Wordpress, Drupal, CakePHP, Laravel ou qualquer outro software que use PHP.

O que me chamou a atenção nesta hoepedagem foi o software de gerenciamento de arquivos, que me permitiu enviar o zip do Joomla de uma única vez.

Geralmente as hospedagens free não permitem enviar um arquivo além de 5MB.

Estou fazendo isso por conhecer as dificuldades de se encontrar uma boa hospedagem free.

Esta permite que utilizemos nosso domínio ou que usemos um subdomínio.

Hospedagem

https://www.000webhost.com/

Principais características:

- 1GB de espaço em disco
- 2 bancos do MySQL
- 5 e-mails
- Hospedagem de domínio ou subdomínio


SSL no Administrator

Como vi que o subdomínio roda com HTTPS, suspeitei de que meu domínio poderia usar também HTTPS. Como só me interessa isso no administrator, experimentei e funcionou. Agora meu administrator usa HTTPS.
Isso valoriza ainda mais esta hospedagem.

Problema

Quando criei meu site para lá e mudei meu domínio tive um problema: não conseguia mais acessar com ribafs.org, somente com www.ribafs.org.

Depois de uma rápida pesquisa encontrei a solução e agora acessa com apenas ribafs.org:

Adicionar ao .htaccess do raiz do site:

RewriteEngine On
RewriteCond %{HTTP_HOST} ^www\.ribafs\.org$ [NC]
RewriteRule ^(.*)$ http://ribafs.org/$1 [R=301,L]

https://julianoaugusto.com/noticias/33-programe-seu-site-para-nao-usar-www

Alerta: não é uma hospedagem para sites com muito tráfego. Este site estava no 000webhost e travava algumas vezes. Então resolvi pagar uma hospedagem e tirar meu site dele.
