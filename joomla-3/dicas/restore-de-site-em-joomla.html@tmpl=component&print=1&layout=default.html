Após o Backup do Site

Após efetuar o backup do site usando o SimpleBackup (https://github.com/ribafs/com_simplebackup), efetue o restore assim:
Após efetuar o backup full do site com o SimpleBackup temos dois arquivos, um com o script completo do banco e outro com o pacote zip com todos os arquivos do site.

É prudente efetuar uma instalação do site logo após backup do mesmo.

Vamos supor que iremos instalar o site localmente.

Para isso precisamos seguir os passos:

- Criar o banco usando um cliente do MySQL

- Importar o script sql gerado pelo backup no banco acima

- Criar uma pasta no diretório web. Exemplo "joomla"

- Descompactar o arquivo zip gerado pelo backup na pasta acima

- Editar o scripr configuration.php e ajustar as configurações:
public $host = 'localhost';
public $user = 'root';
public $password = '';
public $db = 'joomla';

e
public $log_path = '/var/www/html/joomla/administrator/logs';
public $tmp_path = '/var/www/html/joomla/tmp';

Com isso o site está pronto para uso.

http://localhost/joomla/administrator

Use a mesma senha e login que usava no servidor.
