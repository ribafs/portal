Criação de Página de erro (404) para Joomla


Para o caso de o visitante procurar uma página que não existe, então tratemos isso com uma mensagem adequada

Criar um artigo contendo um texto para o visitante que se depara com uma página 404

Título - Página não encotnrada - 404 (outro outro título desejado)
Alias - 404
Categoria - Uncategorized
Opções de Metadados - No index, no follow


Criar um menu oculto

Título - Página não encontrada
Nome único - hiddenmenu
Alias - 404
Opções de Metadados - No index, no follow

Adicionar item de menu para o artigo criado no início

Criar arquivo error.php

templates/seutempalte/error.php

Contendo:

defined( '_JEXEC' ) or die( 'Restricted access');

if ($this->error->getCode() == '404') {
header("HTTP/1.0 404 Not Found");
echo file_get_contents(JURI::root().'404');
exit;
}
