Instalar Extensões Nativas e de Terceiros

Uma extensão simples e importante é a Breadcrumbs.

O Joomla já vem com ela, é nativa.

Extensões - Módulos

Clique em Breadcrumbs

Exibir Título - Ocultar

Salvar e Fechar

Exibindo no Site

Para exibir no site precisamos adicionar ao Template

Extensões - Temas - Portal2 - Layout

Mudar de Portal2 para Base Outline para desbloquear

Agora devemos escolher uma região adequada para o Breadcrumbs. Vamos adicioná-lo logo abaixo da região Navigation. Para isso basta clicar no sinal de + acima de Navigation para que ele adicione uma linha abaixo de si.

Acima e à esquerda, em Positions, clicar e arrastar um Module Instance e soltar na linha criada abaixo de Navigation.

Agora é hora de customizar o Module Instance. Clique na roldana à direita e renomeie (no lápis) para Breadcrumbs. Clique em Pick a Module, selecione Breadcrumbs na lista e clique em Select e então em Apply and Save

E Save Layout

Veja que ele ocupa toda a linha, sem necessidade. Então vamos adicionar outra extensão nativa, a Busca, na mesma linha. Então poderemos escolhar a largura de cada uma simplesmente arrastando.

Adicionar Busca

Extensões - Módulos - Novo - Busca

Título - Busca

Exibir Título - Ocultar

Posição - position-6 (pode ser qualquer uma, pois o Template é quem vai definir)

Exibir pelo Template

Extensões - Temas - Portal2 - Layout (mudar para Base Outline)

Clicar em Module Instance, arrastar e soltar sobre Breadcrumbs, logo à direita. Veja que ele divide em partes iguais (50% para cada).

Podemos alterar as larguras posando o mouse na separação e arrastando para um dos lados. Mudei o BC para 70% e Save Layout.

Clique na roldana à direita do Module Instante da direita, renomeie para Busca e escolha o módulo Busca.

Instalar Extensões de Terceiros

Inicialmente vamos instalar um pequeno componente que criei, que ajuda muito na segurança do site, pois é um componente que efetua backup do site completo, arquivos e banco com muita facilidade. Faça o backup e guarde bem guardado. Sempre que alterar o site efetue novo backup. Ele guarda com a data nos arquivos.

Instalar Componente Simple Backup

Extensões - Gerenciar - Instalar

https://github.com/ribafs/com_simplebackup

Clique em Instalar de uma URL e copie e cole a URL abaixo

https://github.com/ribafs/com_simplebackup/archive/master.zip

Com isso agora temos um novo item em Componentes, que é o Backup Simples.

Vamos efetuar o primeiro backup do nosso novo site:

Componentes - Backup Simples

Clique em Enviar e aguarde que ele mostrará os dois links Arquivos e Banco de Dados. Clique em cada um e guarde bem os arquivos.

Ao instalar uma extensão de terceiros, o Joomla não atribui nenhum item de menu a ela por default. Portanto ela não irá aparecer no site. Requer que você atribua itens de menu à mesma, não esqueça. Geralmente atribuo todos.


Instalar um Gerenciador de Arquivos

Instalarei o extPlorer

Extensões - Gerenciar - Instalar

Faça o download de - http://extplorer.sourceforge.net/

Clique em Enviar Pacote de Arquivos

Indique o arquivo baixado e pronto.

Agora temos um gerenciador de arquivos completo no administrator do site: copiar, criar arquivo ou diretório, upload, download, etc.
