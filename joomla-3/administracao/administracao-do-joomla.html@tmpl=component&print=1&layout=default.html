Administração do Joomla 3.7.2

Através do site administrator criamos o site da frente, adicionamos conteúdo, instalamos extensões, templates, etc.

Antes de iniciar a administração do site, vamos efetuar uma nova instalação do Joomla na pasta portal2, mas agora sem nenhum conteúdo de exemplo, para que seja menos trabalhosa a criação do site.

A instalação será inteiramente semelhante à anterior, exceto que sem conteúdo de exemplo.

Mudar o idioma para pt_BR no site e na administração

Extensions – Manager – Install Languages

 

Digite br na caixa de busca e tecle Enter.

Aparecerão os dois idiomas acima. Clique em Install à esquerda de Portuguese Brazil para instalar e aguarde um pouco.

 

Agora clique em

Extensions – Languages

Clique na estrela na linha de Português do Brasil, à direita para torná-lo o idioma padrão.

Depois clique na combo Site e selecione Administrator e então clique novamente na estrela na linha de Português do Brasil.


Tudo em português do Brasil.

Ativar o mod_rewrite

Clicar no menu

Sistema – Configuração Global

Em Site mude Usar mod_rewrite do Apache para Sim

Agora renomeie o arquivo do raiz do site

htaccess.txt para .htaccess

Obs.: arquivos iniciados com ponto no Linux são arquivos ocultos. Após renomear eles desaparecem. Para que apareçam no modo gráfico tecle Ctrl+H e no terminal usando o comando “ls -la”.

 

Ativar o cache

Clicar em

Sistema – Configuração Global

Na aba Sistema em Cache mude para “Ligado – Cache Progressivo” ou Cache Conservador

Conservador – pequeno cache

Progressivo – mais rápido e grande cache. Não apropriado para sites muito grandes.

 

Instalar um novo Template

Instalar o template jl_learnmate_free - http://www.joomlead.com

https://joomlead.com/joomla-template/learnmate/

Como ele requer o Framework Gantry, vamos instalar antes o Gantry, apenas o componente

http://gantry.org/downloads#joomla

 

Extensões – Gerenciar – Instalar

Clique em Instalar de uma URL e cole a URL do componente (que agora é esta)

https://github.com/gantry/gantry5/releases/download/5.4.13/joomla-pkg_gantry5_v5.4.13.zip

 

Agora então instalar o template learnmate.

Extensões – Gerenciar – Instalar

Faça o download e instale o pacote, pois ele requer seus dados para baixar.

Caso tenha dificuldade de baixar do site oficial, baixe daqui:

http://www.joomla24.com/index.php?option=com_search&Itemid=4&searchword=learnmate

Mas veja que no joomla24 a versão é a 1.0.1 e no site oficial já é a 1.0.2. Prefira do site oficial.

Extensões – Gerenciar – Instalar

Clicar em Enviar Pacote de Arquivos

Aponte para o arquivo baixado clicando no botão procurar pelo arquivo.

Agora instalamos um novo template e temos dois templates para o site. Existem templates para o administrator também, mas não lidarei com eles aqui.

Vamos tornar este template instalado o template padrão do site.

 

Extensões – Temas

Clique na estrela à direita de Learnate para para torná-lo padrão

Agora ele é o nosso tempalte padrão.

Mas precisamos realizar mais uma operação para que o conteúdo do nosso site seja assumido por este novo tempalte: Assignments

 

Criar um Novo Template com o Framework

Como este é mais que um template, mas é um Framework para a criação de templates, então vamos criar um template exclusivo para o nosso site, com o mesmo nome, portal2.

Clique acima em Outlines

Clique no grande sinal de mais (+)

Na janela que se abre em

Title – Portal2

Create Form – Preset

Preset – Default

Confirm

Saia do Gantry. Vá para Modulos, por exemplo.

Agora clique em Extensões – Temas

 

Torne o Portal2 o default.

Clique no Portal2 para acessá-lo

Clique acima em Assignments

Clique no All verde acima para que selecione todos os menus do template.

Depois clique em Save Assignments.

Veja acima e à esquerda que está selecionado Portal2.

Depois de instalar o novo template, torná-lo default e atribuir os menus ao mesm o nosso site .

Iremos remover o que não desejamos e adicionar o que queremos.

Veja que ele traz o main menu (HOME) e também um top menu, que iremos aproveitar para mostrar alguns itens sem subitens.
