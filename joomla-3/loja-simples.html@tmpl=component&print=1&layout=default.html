Loja Virtual Simples para Joomla

Imprimir

Sem Virtuemart, apenas Joomla


Criando uma pequena loja virtual somente com o Joomla, que permita pagamento direto via depósito ou transferência bancária, mas também que ofereça opções de compra pelos cartões de crédito Visa e MasterCard com o PayPal ou através do PagSeguro. Com botão de compra e Carrinho.

Os softwares de loja virtual atualmente apresentam dificuldades, são complexos, especialmente para o gerenciamento da loja. Uma loja com o Joomla tem como grande virtude a facilidade de gerenciamento dos produtos, fazemos isso como se estivéssemos apenas criando um artigo.

Este projeto cria uma loja virtual simples apenas com o Joomla 3.8 sem nenhuma extensão de terceiros. Instalei o Joomla somente para a loja com o template default.

Observações:

Instalar o Joomla do zero sem conteúdo de exemplos.
Instalar e tornar padrão o idioma pt-BR

Temos duas alternativas:
- Criar uma loja virtual em um site existente. Neste caso criamos uma categoria chamada loja e neta todo o conteúdo da mesma
- Instalar o Joomla inteiramente para a loja, então as categorias dirão respeito a cada produto ou categorias de produtos

Vou considerar uma instalação inteiramente dedicada à loja.


Considerarei uma Sapataria com algumas categorias de produtos (estas serão as categorias):
- Tênis
- Sapato social
- Meias
- Mochila

Considerarei 3 formas de pagamentos:
- Depósito ou transferência bancária
- PagSeguro
- PayPal, que aceita cartões VISA e MasterCard

Sugestões:
Também podemos adicionar um link de vídeo sobre o produto na página principal do mesmo.

Assim podemos criar todos os artigos da loja.
Uma forma fácil de criar um artigo é abrindo ele no gerenciador de artigos e efetuando uma cópia. Então mudamos a categoria, se for o caso, o alias, as imagens, preço, descrição, etc.

Geração do Código dos Produtos
Lembre de que cada produto deve ter seu botão criado no PagSeguro, pois tem um código próprio e que não deve ser alterado.
