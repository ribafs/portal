Monaca


É uma IDE online para a criação de aplicativos mobile para Phonegap, Cordova, Ionic e outros.
Compatível com Android, iOS e Windows 8.
Possui ferramentas para debugar o código.
Registro gratuito ou pago. O gratuito permite até 3 projetos.

A maior vantagem que encontrei no monaca foi o fato de não ter que instalar nada. Em algumas ferramentas a trabalheira de instalar e configurar dependências é cansativa. Agradeço à equipe do Monaca.

http://monaca.io

Monaca CLI
sudo npm install -g monaca

Usando

monaca login
monaca create PROJECT_DIRECTORY
Selecione o template com a seta e Enter


Build

Acesse o site, faça login e selecione o projeto e abra
Config - Android KeyStore Setting
Clique em Clear and Generate new
Preencha seu login e senha
Clicar em Android à esquerda (Abaixo de Build & Build Settings)
Clique acima em Build for Release
Clicar no botão abaixo Start Build e aguardar
Clique em Download to Local PC
