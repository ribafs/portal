Monaca - Exemplo

Criação de um app Mobile na IDE online Monaca

http://monaca.io

Vamos criar um aplicativo mobile usando WebSQL. Uma pequena agenda partindo do example1 daqui:
https://github.com/cesarjr/crud-offline

Fazer o download abrir example1.

Acessar o Monaca e efetuar seu registro

Efetuar login

Criar um novo projeto clicando em Create Project

Escolher o template:

JS No Framework - Blank

Clicar em Create New à direita de Blank

Project Name - Agenda
Description - Agenda de telefones

Make Project

Agora Agenda está listada no painel da esquerda

Abrindo o projeto

Clique em Open à direita de Agenda

Agora ele abre o projeto na IDE.

Descrição das regiões da IDE do Monaca

À esquerda - árvore de pastas e arquivos do projeto
Ao centro - o arquivo index.html é aberto no editor de código
Abaixo - a região de debug, logs, warnings e erros
Acima - um menu com algumas opções: Preview (mostra/oculta o Preview), Run on device, etc
À direita - painel de preview da aplicação (um emulador)

Agora vamos excluir o que não iremos usar no projeto do monaca e vamos enviar por Upload os arquivos da agenda.
Também precisamos excluir do index.html do monaca o que não usaremos e copiar e colar do index.html da agenda para o do monaca.

Ao final devemos criar uma chave/certificado para efetuar o Build para Android e podermos publicar no Google Play
- Config - Android KeyStore Settings
- Clear and Generate New

Agora vamos efetuar o Build para Android
- Build - Build App for Android
- Build for Release
- Entrar com a senha da chave
- Clicar em Start Build
Aguardar


- Efetuar o download do APK para o computador
Renomear removendo os números do início e adicionando "agenda".

Finalmente o executável da agenda

agenda.apk
