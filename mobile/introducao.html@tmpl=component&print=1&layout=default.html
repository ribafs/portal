Com o aparecimento do HTML 5, CSS 3 e JavaScript com novos recursos, apareceu uma grande quantidade de frameworks que cria aplicativos e jogos partindo do HTML 5.

Para alguns tipos de aplicativos, o mobile é o mais prático.

Um bom exemplo para a necessidade de um aplicativo é um órgão ou empresa que fornece a sua relação de telefones e ramais em um pdf oferecido em sua página.

Neste caso um pequeno aplicativo de agenda é uma solução mais prática.

Outro detalhe sobre os aplicativos mobile é que eles contam com bancos de dados do lado do cliente, ou seja, podemos criar uma agenda que usa um banco de dados do lado cliente e assim não há necessidade de acessar a internet para consultar a agenda.
