IndexedDb

O indexedDb é uma forma de persistência local do HTML 5. Ele é suportado pela maioria dos navegadores.

O WebSQL é mais popular entre os desenvolvedores por ser SQL, mas o WebSQL foi rejeitado pelo W3 Consórcio e por alguns navegadores.
