Atualizar o Node.JS

No Linux Ubuntu

sudo apt-get update
sudo apt-get install build-essential checkinstall libssl-dev

node -v
sudo npm cache clean -f
sudo npm install -g n

Instalar último node.js
sudo n stable

Anote a versão instalada acima e use abaixo no lugar de 8.2.1:
sudo ln -sf /usr/local/n/versions/node/8.2.1/bin/node /usr/bin/nodejs
node -v

Para desinstalar:
sudo apt-get install --reinstall nodejs-legacy # corrige /usr/bin/node
sudo n rm 8.2.1 # troque pela versão que você instalou
sudo npm uninstall -g n

No Windows

node -v
npm cache clean -f
