Phaser em Mobile

Configurando um jogo para web criado com o Phaser para que se adapte automaticamente em dispositivo móvel.

Adicionar estas 3 linhas de escala ao início do método preload():

preload: function(){
game.scale.scaleMode = Phaser.ScaleManager.SHOW_ALL;
game.scale.pageAlignHorizontally = true;
game.scale.pageAlignVertically = true;

Com isso a visualização pelo navegador já muda. Ele enche a tela se adaptando ao seu tamanho.

Como verificar com facilidade, se realmente se adapta aos dispositivos móveis?
Uma forma simples é instalar a extensão Ripple, que é emulador mobile.

Baixar o emulador Ripper para o Chrome

Acesse com o Chrome
https://chrome.google.com/webstore/detail/ripple-emulator-beta/geelfhphabnejjhdalkjhgipohgpdnoc

Instale a extensão

Agora abra o endereço do jogo pelo navegador Chrome, mas somente com o servidor web:
http://localhost/labirinto

Clique no pequeno ícone azul acima e à esquerda
E Habilite para Phonegap

Finalmente, podemos pensar em instalar nosso jogo em celulares.
Para isso conheço dois serviços simples:
http://build.phonegap.com
http://monaca.io

Precisamos criar um projeto em um destes e adaptar o projeto criado para o código do nosso jogo.
